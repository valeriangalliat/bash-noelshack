NoelShack Upload
================

Upload a picture on [NoelShack](http://www.noelshack.com/).

Description
-----------

Uses cURL to upload a picture with [NoelShack API page](http://www.noelshack.com/api.php).

Dependencies
------------

- `curl`

Options
-------

1. File to upload

Example
-------

```sh
noelshack-up foo.png
```
